# Kroki implementacji (dla Codex LLM / dewelopera)

1. Utwórz strukturé projektu: załód z folder landing/ z plikami index.html, style.css oraz podfolderami img/ i icons/. Skopiuj do img/ wybrane rzechywiste zdjecia magazynu (np. IMG_0772.jpeg, IMG_0729.jpeg, IMG_1588.jpeg - do wyboru podczas implementacje). Umieszć wygenerowane ikony w icons/.  
2. Zdefiniuj paletę kolorów i typografie: podierz z CSS strony glównej Schowane.pl (można uzyć narȩdzi DevTools) wartość kolorów i czcionek. Zapisz je jako zmieenne CSS (--primary-color, --secondary-color, --text-color, --background-color).  
3. Zaprojektuj szkielet HTML: w index.html utworz sekcie zgodnie ze struktura. Uzywaj semantycznych znaczników (<header>, <nav>, <section>, <article>, <footer>). Dodaj identyfikatory (id="hero", id="benefits" itd.), aby uzmgliwo linkowanie w nawigacje.  
4. Dodajtresci: wypeñnij tekstasy zgodnie z podanymi powyzej propozycjami. Zachowaj hierarchi nagłówków (<h1>, <h2>, <h3>). Umieszcz)djecie magazynu w sekcj hero i podpisz je w atrybucie alt opisem ("Widok wnétrza magazynu Schowane.pl").  
5. Osadź wygenerowane ikony: w sekcjach „KorzySci" i „Jak to działa" wstaw obrazy w formacie SVG/PNG. Upewnij są, ze majority jedergowy rozmiar (np. 64x64 px) i są responsywne (height:auto; width:64px).  
6. Zaprojektuj responsywny uklad: uzyj flexbox lub CSS Grid, aby elementy sekcji ukladaly sie w kolumny na desktopie i w jeder kolumne na urzadzeniach mobilnych. Zadbaj o marginesy i paddingi zgodne z estetyką strony glówne Schowane.pl.  
7. Stylizacja CTA: wyróznij przyciski CTA poprzej uzycie koloru glówno i zaokraglenia (np. border-radius: 6px). Dodaj efekt hover (background-color lekko ciemniejszy, transform: scale(1.05)).  
8. Dodaj animacje i interakcie (opcjonalnie): uzyj prostych animaci CSS (np. fade-in dla sekci podczas scrollowania) oraz anchor links w nawigȩ, któ rplynnie scrolluju do odpwiednich sekci( scroll-behavior: smooth).

9. Testy i optymalizacja: sprawdź, jak strona wygląda na rożnych rozdzielczośćciach. Upewnij są, ze obrzy są zoptymalizowane (rozmiar  $< 500$  kB). Dodaj meta-tags SEO (tytu#, opis, open graph) opisuje oferte przechowalni.  
10. Finalne sprawdzenie: zweryfikuj,czy wzystkie linki dzialaja, przyciski CTA prowadzao do formularza/sekcj "Kontakt", a tresci sa poprawne językowo. Popró zespół o ostateczna akceptacja. Po zaakceptowaniuCnoza wdrożć stroné na serwerze Schowane.pl (np. jak statyczny plik HTML z wąsnym routingiem).

Ten plan stanowy szczegółowy przywodnik dla zespolu deweloperskiego (np. Codex LLM) w celu zaprojektowania i wdrożenia spójnégo, konwertujęcego landing page, zgodnégo z marka Schowane.pl.
